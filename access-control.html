ABAC Conditions

Attribute rules evaluated at decision time. Each rule is authored against a resource type, which determines the resource.* attributes its builder offers (generic rules see only owner). Assign one or more to a grant by clicking a matrix cell; when a grant carries several they must all pass. The In use count is how many grants reference each rule.

Users

Resource Types

Resource types are the classes of protected object that roles get permissions on. Each defines its own actions and its own set of attributes (exposed to ABAC rules as resource.<name>).

Resource Items

Concrete instances of a resource type. Each item has an owner — the subject checked by the If owner condition at decision time — and its own attribute values for the attributes its type declares (blank inherits the type default). A type itself has no owner.
